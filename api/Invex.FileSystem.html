Invex.FileSystem
Table of Contents

Namespace Invex.FileSystem

Classes

FileSystemHostExtensions

Provides extension methods for registering the file system services with an IServiceCollection.

FunctionPathProvider

A lightweight, delegate-based implementation of IPathProvider that resolves paths using a caller-supplied function rather than requiring a dedicated class.

RootedPath

Represents a file system path that is rooted within a specific System.IO.Abstractions.IFileSystem instance.

TransformFileScope

Provides disposable scopes for performing temporary, reversible transformations on a single file's content.

TransformFileScopeExtensions

Provides extension methods for fluently chaining additional transformations onto a Task<TResult> that yields a scope returned by one of the static CreateAsync factory methods.

TransformMultiFileScope

Provides disposable scopes for performing temporary, reversible transformations on multiple files simultaneously. Upon disposal, the original content of every managed file is restored unless restoration is explicitly cancelled.

Interfaces

IPathMarker

Defines a contract for types that represent a specific, well-known file path within the file system, resolved relative to the current System.IO.Abstractions.IFileSystem instance.

IPathProvider

Defines a provider that can resolve a RootedPath for a given key within the file system.

IRootedFileSystem

Defines a specialised file system abstraction, layering build-aware path resolution on top of the standard System.IO.Abstractions.IFileSystem abstraction.
