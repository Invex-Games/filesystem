Interface IPathProvider

Namespace

Assembly
Invex.FileSystem.dll

Defines a provider that can resolve a RootedPath for a given key within the file system.

[PublicAPI]
public interface IPathProvider

Remarks

Multiple providers can be registered simultaneously. GetPath(string) queries them in descending Priority order and returns the first non-null result. A provider that does not recognize a key should return null to let the next provider in the chain handle it.

Because GetPath(string) caches successful resolutions, a provider's GetPath(string) method is called at most once per key per lifetime of the IRootedFileSystem instance.

Properties

Priority

Gets the priority of this provider.

int Priority { get; }

Property Value

int

Remarks

Higher values are queried before lower values. When two providers can both resolve the same key the one with the higher priority wins. Use a negative value to register a low-priority fallback that only applies when no other provider matches.
